# Oladipupo Store Data Analysis (Interactive Dashboard creation )
## Project Objective
Oladipupo Store wants to create an annual report for Maintenanace for 2022, 2023 and 2024, to analyse all maintenances made.
## Dataset used
https://docs.google.com/spreadsheets/d/1CAT5kcz5HGAEOz4lmiokEmLFbbs6cQQI/edit?usp=drive_link&ouid=104177868748180292987&rtpof=true&sd=true
## Process
- Verify data for any missing values and anomalies, and sort out the same .
- Made sure data is consistent and clean with respect to data type, data format and values used.
- Created tables and charts according to the questions asked.
- Merge all  tables into one dashboard and apply slicer to make dynamic.
## KPIs
- Total Maintenance cost
- Total Downtime Hours
- Total Part Cost
- Total Labor cost

 ## Project Insight
- Maintenance costs are rising steadily — there’s a clear year-over-year increase, with 2024 being the highest contributor.
- Downtime is heavily concentrated in a few truck brands — Peterbilt and Freightliner lead in downtime hours.
- Parts cost dominates spending — at 4.45M, it’s significantly higher than labor (1.28M), suggesting parts are the main cost
 driver.
- Labor hours are uneven across makes — Peterbilt and Freightliner again top the list, indicating they may require more intensive servicing.
## Conclusions
- A few truck brands are driving most of the problems — especially Peterbilt and Freightliner across downtime and labor.
- Cost control should focus on parts management — since parts make up the bulk of expenses.
- Maintenance strategy may need review — rising costs and concentrated downtime suggest inefficiencies or aging assets.
